# quera scrapper

scraping all [quera job opportunity](https://quera.ir/magnet/jobs) and give you some information about programming and technologies job market

## Usage

if you run this application you will see :

- available jobs count
- remote jobs count and percent of remote jobs
- full time , part time and project base jobs and their percent of them
- count of intern , junior , senior and lead developers and their percent of them who jobs want them
- 3 of most popular main and normal technologies

## Installation and run

1. install python and pip
2. clone/download main.py and requirments.txt
3. install requirements

```bash
pip install -r path/to/requirments.txt
```

or

```bash
pip3 install -r path/to/requirments.txt
```

4. run code and wait
